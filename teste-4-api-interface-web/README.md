# Teste de Processo Seletivo — IntuitiveCare

Este repositório contém os testes desenvolvidos como parte de um processo seletivo para vaga de estágio na IntuitiveCare.

Os testes abordam diferentes etapas de um fluxo de dados, desde integração com API pública, transformação e validação de dados, uso de banco de dados, até a criação de uma API e interface web para visualização das informações.

---

## Estrutura do repositório

- `teste-1-integracao-api/`  
  Integração com API pública da ANS, download e consolidação de dados.

- `teste-2-transf-validacao-dados/`  
  Transformação, validação, enriquecimento e agregação dos dados.

- `teste-3-banco-dados/`  
  Scripts SQL para criação de tabelas, importação dos dados e consultas analíticas.

- `teste-4-api-interface-web/`  
  API em Python (Flask) e frontend em Vue.js para visualização dos dados.

---

## Observações

- Este repositório é público apenas para fins de portfólio e avaliação do processo seletivo.
- O código foi desenvolvido com foco educacional e demonstração de conhecimentos práticos.
- Cada pasta de teste possui (ou deve possuir) seu próprio README com instruções específicas de execução.

---

## Como executar

Cada teste possui instruções próprias dentro de sua respectiva pasta.  
Recomenda-se consultar o README de cada diretório para entender como executar cada parte do projeto.
